Quick Start

Get up and running in under 2 minutes.

# 1. Clone and enter the project
git clone <your-repo> && cd groq_testing

# 2. Create virtual environment
python -m venv .venv && source .venv/bin/activate

# 3. Install dependencies
pip install langchain langchain-groq langchain-huggingface fastmcp groq mcp python-dotenv loguru

# 4. Create .env file
echo "GROQ_API_KEY=your_key_here" > .env

# 5. Run your first chain
python basic_langchain_groq.py

# 6. Start the FastMCP server
python fastmcp_server.py

# 7. In another terminal, call the server
python fastmcp_client.py
